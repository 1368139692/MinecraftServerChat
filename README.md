# 介绍:  
MinecraftServerChat是一个公益项目  
此项目的初衷仅仅是为了能让服务器信息对外开放  
让新玩家在不进入服务器的前提下  
更快更全面的了解服务器给玩家留下一个好印象  
给各位服主们提供一个精简的官网/社区
